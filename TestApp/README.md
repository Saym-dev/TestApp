# TestApp

Тестовое iOS-приложение с онбордингом, paywall и эмуляцией подписки.

## Описание

Приложение демонстрирует простой пользовательский флоу:
- Онбординг
- Экран выбора подписки (месяц / год)
- Главный экран

## Логика флоу
- При первом запуске показывается онбординг
- После онбординга открывается paywall
- После "покупки" подписки пользователь попадает на главный экран
- Статус подписки сохраняется между запусками приложения
- При повторном запуске, если подписка активна, сразу открывается главный экран

## Технологии
- Swift
- SwiftUI
- AppStorage (UserDefaults)
- NavigationStack

## Запуск проекта
1. Открыть проект в Xcode
2. Выбрать iOS Simulator
3. Запустить приложение (Cmd + R)
